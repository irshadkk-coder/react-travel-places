# React Travel Places

A simple React frontend project that displays travel places by fetching data from an API.

## Features

* Fetch places from API
* Display places in a grid layout
* Click a place to view detailed information
* Image gallery for each place
* Responsive UI

## Tech Stack

* React
* React Router
* JavaScript
* CSS
* Fetch API

## Installation

Clone the repository

git clone https://github.com/irshadkk-coder/react-travel-places.git

Go to project folder

cd react-travel-places

Install dependencies

npm install

Run the project

npm start

The app will run on:

http://localhost:3000
